## 2. 里程碑（阶段 S0～S4）

> 下表 **S0～S4** 为本模块 **交付阶段代号**，勿与「模块 M1」混淆。

| 阶段 | 交付要点 |
|------|-----------|
| **S0** 基线 | 工程骨架、CI、`research_messages` / `reads` 迁移、**种子数据** |
| **S1** 列表 | `GET /messages` 游标分页 + Inbox 页（`05` US-001） |
| **S2** 详情与已读 | `GET /{id}` + `PATCH /read` + 详情路由（US-002） |
| **S3** 搜索 | `GET /search` + 搜索页 + 限流（US-003） |
| **S4** 发布准备 | `07` 指标与告警接入、压测报告、UAT |

## 3. 任务拆分（WBS 摘要）

| ID | 任务 | DoD |
|----|------|-----|
| M1-W1 | DB 迁移 + seed | 符合 `10` §2 |
| M1-W2 | Flask `research_bp` 五端点 | 符合 `09` §3 |
| M1-W3 | React 三路由 | 符合 `06` §2～3 |
| M1-W4 | pytest + E2E 抽样 | 覆盖 `05` P0 US |
| M1-W5 | SLS/指标 | 符合 `07` §3 |

## 4. 任务与依赖

- **首版不依赖** M2/M4 服务在线；M2 接入后 **可删 seed 依赖**，不改 `09` 读契约。  
- 与全局鉴权契约对齐（登录态）。

## 5. 风险与应对

| 风险 | 应对 |
|------|------|
| 早会高峰读多 | 只读副本 / 缓存首屏（`07` 降级） |
| 搜索慢 | `tsvector` 或限制 `limit`、加索引（`10` §3） |
